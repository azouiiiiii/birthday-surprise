# 🎂 Birthday Surprise

一个基于 **React + Three.js** 构建的沉浸式 3D 生日互动网页。  
通过重构经典 3D 场景结构，将“节日氛围 + 个性化记忆展示”结合，打造更具情感表达的互动体验。

---

## 🌐 在线预览

👉 https://smallai-react-3d-prompt.vercel.app

---

## ✨ 项目亮点

- 🎂 **结构重构设计**  
  将原项目中的 Christmas Tree 场景重新设计为“蛋糕模型”，更贴合生日主题  

- 💥 **动态拆解与重组动画**  
  蛋糕在交互触发后会“散开”，并在动画结束后重新聚合  

- 🖼 **照片环绕系统（核心创新）**  
  - 蛋糕散开后触发照片环绕动画  
  - 多张照片围绕中心对象动态分布  
  - 支持点击单张照片进行放大查看  

- 🎵 **多媒体融合体验**  
  音乐、粒子效果与 3D 场景结合，增强沉浸感  

---

## 🧠 设计思路

本项目的核心在于：

> 将“静态祝福页面”转化为“可交互的记忆空间”

通过以下方式实现：

- 使用 3D 场景承载情绪表达  
- 用“拆解 → 展示 → 重构”隐喻记忆展开过程  
- 将照片作为情感载体嵌入交互流程  

---

## 🔄 灵感来源

本项目基于以下开源项目进行二次创作与重构：

- smallai-react-3d-prompt

在此基础上进行了以下修改：

- ❌ 移除了黑洞（Black Hole）模块  
- 🔁 重构 Tree 结构为 Cake  
- ✨ 增加照片交互系统  
- 🎯 聚焦“生日场景”表达
